DOWNLOADING

  1. Download Python 3.10 by going to python.org/downloads and follow the download instructions.
  2. Once Python is downloaded, go to the command prompt and type in py -m pip install PyQt5 and hit enter.
  3. After PyQt5 has finished downloading, type in py -m pip install PyQtWebEngine into the terminal and hit enter
  4. Afterwards, download the browser code using this link. Download

OPENING THE BROWSER

  1. Open the command prompt
  2. Type in the complete file path into the prompt. The default for Windows should be C:\Users\(username abbr)\Downloads\main.py. After that, hit enter and wait a few seconds. The browser should fire up afterwards.

HOW TO USE THE BROWSER

The Top Buttons

  • Use the text bar at the left to type in a URL and hit the "Go" button.
  • Use the text bar to the right to search anything on DuckDuckGo.com. Hit "Search" afterwards.
